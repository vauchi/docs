# Knowledge Documents

This directory contains external research and learnings captured during investigation phases.

## Purpose

When investigating a problem or planning an implementation, any external knowledge gathered should be documented here for future reference. This includes:

- Industry practices and standards
- Prior art research
- RFCs and specifications
- Academic papers summaries
- Tool/library evaluations
- Security research

## Naming Convention

`YYYY-MM-DD-<topic>.md`

## Examples

- `2026-01-23-webrtc-data-channels.md` - Research on WebRTC for real-time communication
- `2026-01-23-x25519-key-exchange.md` - Analysis of X25519 for key exchange
- `2026-01-23-qr-code-capacity.md` - QR code data capacity research
