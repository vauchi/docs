## Quick Reference

### Add a Contact

1. Meet in person
2. Open **Exchange** tab
3. Show your QR code
4. Scan their QR code

### Link a Device

1. Open **Settings > Devices** on existing device
2. Tap **Link New Device**
3. Scan QR on new device

### Hide a Field from Someone

1. Go to **Contacts**
2. Select the contact
3. Toggle the field off

### Create a Backup

1. Go to **Settings > Backup**
2. Tap **Export Backup**
3. Enter a password
4. Save the backup code
